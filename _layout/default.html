Vidconvert

Mass batch conversion using ffmpeg. Converts video files into Mp4 and then sorts them

Compatibility

Right now vidconvert only supports Linux

Install

  • clone repository
  • cd vidconvert
  • sudo make setup

Usage

after install just move inside the directory where you want to convert files to mp4, then type vidconvert.py
